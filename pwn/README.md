# Pwn / Binary Exploitation

**Note:** Some challenge source files and binaries in this section were provided during Offensive Security laboratory courses in my Master’s program at Politecnico di Torino. The analysis, write-ups, and exploit scripts are my own work. These materials are shared for educational and portfolio purposes; flags, credentials, and active challenge endpoints have been removed.
## Challenges

Write-ups will be organized from introductory stack exploitation to advanced heap exploitation.

## Techniques covered

- Stack-based buffer overflows
- ret2win
- Stack-canary bypass
- PIE and libc address leaks
- ROP and ret2libc
- Direct syscall chains
- Use-after-free
- Tcache poisoning
